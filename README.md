# Net Worth Growth Simulator

An interactive wealth growth simulator for modeling your net worth across asset classes, inflation, taxes, withdrawals, and scenarios.

## Features

- 24+ asset categories (stocks, bonds, retirement accounts, real estate, crypto, alternatives, and more)
- 8 liability types with separate net worth tracking
- Inflation adjustment with custom rate
- Three-layer tax modeling (gains tax + withdrawal tax + home exemption)
- Annual withdrawals with prioritized order
- Rebalancing toggle
- Year-by-year breakdown with age, multiple, and YoY %
- Scenario planning with comparison charts
- Three preset allocation strategies (Defensive, Endowment, Aggressive)
- Custom asset categories with user-picked colors
- Auto-save to browser localStorage

## Local Development

```bash
npm install
npm run dev
```

Then open http://localhost:5173
